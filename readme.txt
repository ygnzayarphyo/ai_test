For question number 1

Step 1: Environment Setup...
        install php from this guide https://laravel.com/docs/4.2

Step 2: go under q1/ and run this script
        > php artisan serve

Step 3: Launch one of the following url...
        - for localhost -> http://localhost:8000/botman/tinker
        (or)
        - for web hosting -> https://botmanidea.000webhostapp.com/botman/tinker

Step 4: click on bottom right "Message" icon

Step 5: Type `start` command without `` characters in "Send a message..." box.

For question number 3

Step 1: Environment setup...
        install python from this guide https://realpython.com/installing-python/

Step 2: Install necessary libraries..
        > pip install bs4
        > pip install BeautifulSoup
        > pip install cherrypy

Step 3: Run the python script
        > python q3.py

Step 4: Launch the following url at web browser (Chrome, Mozilla FireFox, etc..)
        http://localhost:8080/
